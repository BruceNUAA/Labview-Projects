The following scripts use a pattern template to detect common objects such
as checkers and characters.  

On the character script, the matching character on the right side is not 
detected - possibly due to the character being close to the edge of the
image.

Demo of scripts can be found on:

https://drive.google.com/open?id=0B4ZQNQ2h_OcRTFA2THF5ZVU0Nlk
